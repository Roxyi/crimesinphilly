    Crimes in Philly

    This is a web mapping interface displaying the crimes in Philly during the time period from 2015-01-01 to 2016-04-12. It aims to help the citizens in Philadelphia know more about crimes in the neighborhood and avoid the dangerous corners.

    The data source is Year-to-Date Part I Crime Incidents .

    The data is stored on CartoDB. I mainly used leaflet.js and CartoDB.js to build the web map. Actually, to solve the conflict between CartoDB.js and jQuery, I used cartodb_nojquery.js. It enabled me to display the torque layer.

    An alternative URL of the project is: crimesinphilly.com. I used AWS to host the website. However, for some reason, I cannot use geolocation function on Chrome with an http domain. I am still trying to figure out how to get an https domain. Therefore, if you use the alternative URL to browse the website, you may need to use Firefox to enable the geolocation function.
